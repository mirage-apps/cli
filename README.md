# Mirage CLI

The **Command-Line Interface** for Mirage Stores

## Installation
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/mirage-apps/cli/master/install.sh)"


### Development
**Generate the binary and run binary**:
```cpp
make
```

**Generate the binary**:
```cpp
make binary
```

**Delete the binary**:
```cpp
make clean
```
